# GOST cryptography

Russian block ciphers:

- implement Russian block ciphers Kuznyechik and Magma (GOST 34.12-2015) [1];
- unit tests for functions and encryption/decryption, which follow examples (test vectors) [1].

Mã khối của Liên bang Nga:

- cài đặt các mã khối của Liên bang Nga, bao gồm Kuznyechik và Magma (GOST 34.12-2015), được định nghĩa ở [1];
- unit tests cho các hàm và mã hóa/giải mã theo test vectors được mô tả ở [1].

# Bibliography
